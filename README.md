# Smart Expense Tracker (CLI-Based)

A modular, object-oriented Command Line Expense Tracker built using Python. This application allows users to manage daily expenses efficiently with features like adding, viewing, deletingm annd generating montly summaries. Data is sotred persistently using CSV file handling.

## Features

- Add expenses with date, category, amount, and description
- View all expenses in formatted table
- Delete expenses by ID
- Generate monthly summary with category-wise breakdown
- Persistent storage using CSV files
- Input validation for dates and amounts
- User-friendly menu-driven interface

## Tech Stack

- Python 3.x
- CSV module for data storage
- Datatime module for date operations

## Concepts Implemented

- Object-Oriented Programming (OOP)
- File Handling (CSV)
- Input Validation
- CRUD Operations
- Data Aggregation using Dictionary
- CLI-based User Interface
